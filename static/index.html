CONNECTING ALL
THE BLOCKCHAINS
WITHOUT
ADDING YET ANOTHER ONE

We believe that a good cross-blockchain solution is one
that minimizes complexity instead of adding
to it.
The COMIT open source protocol is designed to be:

PURE  — 

No additional chain. No additional token.

NATIVE  — 

Any blockchain. Any layer.

TRUSTLESS  — 

No trusted 3rd parties. Verify, don’t trust.

ACCOUNTLESS  — 

No need to create an account or deposit tokens.

OPEN  — 

Tools and codebase that are open source.
